# Educational components (`edu/`)

| Directory | Topic |
|-----------|--------|
| `dig/` | Digital logic & representations — bits, Boolean algebra, K-maps, error coding, Verilog |
| `ee/` | General electrical engineering (e.g. circuit diagram builder) |
| `RF/` | Radio-frequency subset — antennas, Smith chart |
| `prog/` | Programming — ISA simulators, C++ execution/optimization visualizer |
| `dstruct/` | Data structures |
| `mech/` | Mechanics (placeholder) |
| `quantumc/` | Quantum computing (placeholder) |
| `docs/` | Shared instructions / help Markdown for the launcher |

Shell UI and shared simulator chrome live under `../util/`.
